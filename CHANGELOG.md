# Версия 0.05

## Изменения относительно v0.04

### Добавлено
- **Быстрый вход** на странице логина — кнопки для всех пользователей
- **Система обратной связи** — форма для отчётов о багах и предложений
- **SOPы для выделения клеток:**
  - Выделение мононуклеаров методом градиентного центрифугирования (Фиколл)
  - Получение макрофагов из мононуклеаров (адгезионный метод)
- **Связь SOP с культурами** — при создании первичной культуры выбирается SOP и запускается запись производства
- **Версия в интерфейсе** — отображение номера сборки и даты
- **Git-репозиторий** для версионирования

### Исправлено
- Баг с пропаданием донаций после добавления
- Баг с не списыванием среды при создании культуры
- Краш страницы "Культуры"
