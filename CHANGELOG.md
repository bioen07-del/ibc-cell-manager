# Версия 0.04

## Изменения относительно v0.03

### Добавлено
- **Модуль SOP (Standard Operating Procedures)**
  - Создание и редактирование стандартных операционных процедур
  - Пошаговое выполнение протоколов
  - Версионирование СОП
  - Связь СОП с задачами
  - Категории: размораживание, пассирование, заморозка, контроль качества, общее
  - Связь с типами манипуляций (thawing, passaging, freezing и др.)

- **Расширение модели Task**
  - Поле `sop_id` — связь задачи с конкретным СОП
  - Поле `sop_execution_id` — связь с выполнением СОП

### Сохранено из v0.03
- Локальное хранилище данных (localStorage)
- Упрощённая архитектура без зависимости от Supabase
- Оптимизация через useMemo
- Конвертер snake_case/camelCase (dataMapper)
- Все существующие модули: доноры, культуры, хранение, оборудование и др.
